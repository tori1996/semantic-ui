Design Beautiful Websites Quickly

Semantic is a development framework that helps create beautiful, responsive layouts using human-friendly HTML.

Concise HTML

Semantic UI treats words and classes as exchangeable concepts. Classes use syntax from natural languages like noun/modifier relationships, word order, and plurality to link concepts intuitively. Get the same benefits as BEM or SMACSS, but without the tedium.

Intuitive Javascript

Semantic uses simple phrases called behaviors that trigger functionality. Any arbitrary decision in a component is included as a setting that developers can modify.
